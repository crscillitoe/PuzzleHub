How to setup a separate user for interacting with the PuzzleHub database:

Step 1:

    - run the following command:
            sudo mysql -p < create_db_user.sql
    
    - this will create a database-specific user with the following credentials:
            Username: puzzleHub
            Password: ch4nge_m3!
        
Step 2:

    - exit mysql

Step 3:

    - run the following command:
        mysql -u puzzleHub -p puzzleDatabase < database_gen.sql
